# Simon-says-Game
This is a simple and fun Simon Says game, where players test their memory by following a sequence of colors and sounds. Each round, the sequence grows, challenging players to remember and replicate it correctly!
Simon Says Game 🎉
A memory game where players follow a growing sequence of colors and sounds. Each round, a new step is added to the sequence—making it progressively challenging!

How to Play
Watch and listen to the sequence.
Repeat the pattern.
Keep advancing as the sequence grows!
Setup
Clone the repository.
Run npm install.
Start with npm start.
Test your memory!
